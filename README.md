## Features

- Provides a command to generate unit tests for your code.
- Displays a list of test titles in a dropdown for user selection.
- Allows users to select a file in their testing directory to get the testing style.
- Provides the generated test in a pop-up window for easy copying and pasting.

## Requirements

- Node.js
- VS Code version 1.95.0 or higher

## Extension Settings

This extension contributes the following settings:

- `unitgpt.enable`: Enable/disable this extension.
- `unitgpt.apiKey`: Set your OpenAI API key.
